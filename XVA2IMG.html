XVA2IMG

Project repo: LINK

Introduction

Extracts disk images from XenServer XVA files.

Convert XVA file to IMG.
USAGE: 
xva2img.py [XVA FILE]

Disk images are extracted into the current directory.

There is an alternative version called "xva2img-extract.py" that will extract the full file structure from the XVA file to a folder before rebuilding the image files. This allows you to examine the XVA contents and perform other tasks after image extraction.
